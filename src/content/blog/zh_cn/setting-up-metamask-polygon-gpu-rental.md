## 为什么选择Polygon而不是Ethereum主网

在设置任何内容之前，了解Polygon的重要性将为您在每笔交易中节省费用。

### gas费问题

每笔区块链交易都需要向网络验证者支付费用。在Ethereum主网上，这些费用（称为"gas"）会根据网络需求波动：

**Ethereum主网gas成本：**

- 简单转账：$1-5
- 代币批准：$2-8
- 复杂交易（托管存款）：$5-15
- 高需求期间：$20-50+

**对于GPU租赁，这会产生问题：**

假设您以$0.60/小时租赁RTX 4090 2小时：

- GPU成本：$1.20
- 存款的Ethereum gas：$5.00
- 提款的Ethereum gas：$5.00
- **总计：为$1.20的GPU时间支付$11.20**

当gas费超过租赁成本时，小额交易在经济上就不合理了。

### Polygon：相同的安全性，费用降低100倍

Polygon是建立在Ethereum之上的"Layer 2"网络。它继承了Ethereum的安全模型，同时以极低的成本处理交易：

**Polygon gas成本：**

- 简单转账：$0.001-0.01
- 代币批准：$0.005-0.02
- 复杂交易（托管存款）：$0.01-0.05
- 高需求期间：$0.05-0.20

**同样的GPU租赁场景在Polygon上：**

- GPU成本：$1.20
- 存款的Polygon gas：$0.02
- 提款的Polygon gas：$0.02
- **总计：为$1.20的GPU时间支付$1.24**

gas费变得可以忽略不计，而不是令人望而却步。

### 技术细节（供好奇者参考）

Polygon通过不同的共识机制和批量交易处理实现低费用。技术架构不如实际结果重要：

| 网络         | 确认时间 | 交易费用     | 安全性             |
| ------------ | -------- | ------------ | ------------------ |
| Ethereum主网 | 15-60秒  | $1-15        | 最高               |
| Polygon      | 2-5秒    | $0.001-0.05  | 高（Ethereum保护） |
| Solana       | <1秒     | $0.0001-0.01 | 高                 |

GPUFlow和其他原生加密GPU平台使用Polygon，因为：

- 费用不会破坏小额租赁的经济性
- 快速确认意味着快速访问
- 通过Ethereum锚定保持强大的安全性
- 广泛的稳定币可用性（USDC、USDT）

### 您的钱包在两个网络上都有效

这是关键概念：**您的MetaMask钱包地址在Ethereum和Polygon上是相同的。**

地址`0x7a3B...4f2D`在两个网络上都存在。发送到Polygon上该地址的资金会留在Polygon上。在Ethereum上发送的资金会留在Ethereum上。相同的地址，不同的网络，单独的余额。

这就是为什么在发送资金时网络选择很重要。在Ethereum主网上向您的地址发送USDC不会使这些资金在Polygon上可用——尽管地址相同，但它们在不同的网络上。

## MetaMask安装和助记词安全

MetaMask是一个浏览器扩展钱包，可存储您的加密货币并允许您与区块链应用程序互动。安装大约需要五分钟。

### 步骤1：下载MetaMask

**直接访问官方来源：**

1. 打开浏览器（Chrome、Firefox、Brave或Edge）
2. 访问**metamask.io**（仔细验证URL——存在钓鱼网站）
3. 点击"Download"
4. 选择您的浏览器
5. 在扩展商店中点击"添加到[浏览器]"
6. 出现提示时确认安装

**警告：**仅从metamask.io或官方浏览器扩展商店下载MetaMask。切勿从电子邮件、Discord消息或搜索广告中的链接安装。假冒的MetaMask扩展程序会窃取资金。

### 步骤2：创建新钱包

安装后：

1. 点击浏览器工具栏中的MetaMask狐狸图标
2. 点击"Get Started"
3. 选择"Create a new wallet"
4. 同意条款（如果愿意可以阅读，标准软件条款）
5. 创建密码

**此密码：**

- 在此特定设备上解锁MetaMask
- 不会在您失去访问权限时恢复您的钱包
- 应该强大但不需要永久记住
- 与您的助记词分开（接下来解释）

### 步骤3：保护您的助记词

**这是最关键的步骤。仔细阅读。**

MetaMask将以特定顺序显示12个单词。这些单词是您的"助记词"或"恢复短语"。

**助记词的作用：**

- 从任何设备提供对您钱包的完全访问权限
- 允许在计算机死机、浏览器崩溃或MetaMask损坏时恢复
- 无法更改或重置——这12个单词对此钱包是永久的
- 任何拥有这些单词的人都可以立即取走您的所有资金

**应该做的事：**

1. MetaMask在屏幕上显示12个单词
2. **按确切顺序将它们写在纸上**（单词1、单词2...单词12）
3. 不要拍照、截屏或输入到任何数字设备
4. 不要存储在密码管理器、云盘或笔记应用中
5. 将纸张存放在安全的地方（保险箱、上锁的盒子、银行保险箱）
6. 考虑制作第二份副本存放在不同的物理位置

**不应该做的事：**

- 切勿在任何网站上输入助记词（合法网站绝不会要求）
- 切勿与任何声称是"支持人员"的人分享
- 切勿以任何形式数字化存储
- 切勿在他人或设备听力范围内大声朗读

**为什么这很重要：**

与银行账户不同，加密货币钱包没有客户支持、欺诈部门或交易撤销。如果有人获得您的助记词：

- 他们会在几秒钟内清空您的钱包
- 任何机构都无法撤销交易
- 无法恢复
- 资金永久丢失

助记词就是钱包。请相应地保护它。

### 步骤4：确认助记词

MetaMask会要求您通过按顺序选择单词来确认：

1. 按正确的顺序点击单词
2. 这可以验证您确实已将它们写下来
3. 不要通过临时记忆跳过此步骤——您需要书面备份

### 步骤5：钱包就绪

确认后，您的钱包已创建并可以使用。您将看到：

- 您的钱包地址（以`0x`开头，后跟40个字符）
- 显示0 ETH的余额（您尚未充值）
- 显示"Ethereum Mainnet"的网络指示器

**您的钱包地址：**

- 这是公开的——可以安全分享以接收资金
- 就像加密货币的电子邮件地址
- 不会改变且无法修改
- 相同的地址适用于多个网络（Ethereum、Polygon等）

## 向MetaMask添加Polygon网络

MetaMask默认设置为Ethereum主网。要使用Polygon的较低费用，您需要添加该网络。

### 方法1：自动添加（最简单）

1. 访问**chainlist.org**
2. 搜索"Polygon"
3. 找到"Polygon Mainnet"（Chain ID: 137）
4. 点击"Add to MetaMask"
5. MetaMask弹窗出现——点击"Approve"
6. 点击"Switch to Network"以激活Polygon

**完成。**Polygon现在可在您的网络下拉列表中使用。

### 方法2：手动添加

如果chainlist.org不起作用，请手动添加：

1. 点击MetaMask扩展图标
2. 点击网络下拉列表（显示"Ethereum Mainnet"）
3. 点击"Add network"
4. 点击"Add a network manually"
5. 输入以下确切详细信息：

```
Network Name: Polygon Mainnet
New RPC URL: https://polygon-rpc.com
Chain ID: 137
Currency Symbol: MATIC
Block Explorer URL: https://polygonscan.com
```

6. 点击"Save"
7. Polygon出现在您的网络列表中

### 在网络之间切换

添加Polygon后：

1. 点击MetaMask中的网络下拉列表（扩展程序顶部中央）
2. 选择"Polygon Mainnet"或"Ethereum Mainnet"
3. 您的余额显示会更改为显示所选网络上的资金

**记住：**相同的地址，不同的网络。在Ethereum上的资金在查看Polygon时不可见，反之亦然。您正在查看同一个邮箱，但在不同的邮局。

### 验证网络添加

确认Polygon正在工作：

1. 切换到Polygon Mainnet
2. 余额应显示"0 MATIC"（您尚未充值）
3. 网络名称旁边出现小的Polygon标志
4. 地址与Ethereum上的相同

如果您看到有关RPC连接的错误：

- 尝试备用RPC URL：`https://rpc-mainnet.matic.quiknode.pro`
- 或：`https://polygon-bor.publicnode.com`
- 网络拥塞偶尔会导致RPC问题

## 使用USDC为钱包充值

您的钱包存在但什么都没有。本节介绍如何将USDC（推荐用于GPU租赁的稳定币）存入您在Polygon网络上的MetaMask钱包。

### 为什么GPU租赁使用USDC

**稳定币消除价格波动：**

- USDC与美元保持1:1挂钩
- 今天的$100 USDC等于明天的$100 USDC
- 加密货币价格波动不会影响您的GPU预算

**与波动性加密货币比较：**

- 为GPU租赁存入$100的ETH
- ETH一夜之间下跌10%
- 您剩余的余额只能购买少10%的GPU时间
- 价格在您没有采取任何行动的情况下对您不利

**USDC完全避免了这一点。**无论市场状况如何，您的GPU租赁预算都保持稳定。

### 充值方法1：从交易所提款（推荐）

如果您已经在交易所（Coinbase、Binance、Kraken等）拥有加密货币，直接提款到Polygon是最便宜的选择。

**Coinbase分步指南：**

1. 登录Coinbase
2. 导航到USDC余额
3. 点击"Send"
4. 输入您的MetaMask钱包地址（从MetaMask复制）
5. **关键：**选择"Polygon"作为网络
6. 输入要发送的金额
7. 查看费用（Polygon提款通常为$0.10-1.00）
8. 确认提款
9. 等待确认（通常1-10分钟）

**Binance分步指南：**

1. 登录Binance
2. 导航到Wallet → Spot Wallet
3. 找到USDC，点击"Withdraw"
4. 粘贴您的MetaMask钱包地址
5. **关键：**选择"MATIC"或"Polygon"网络
6. 输入金额
7. 完成安全验证
8. 等待确认（通常5-15分钟）

**支持Polygon USDC提款的常见交易所：**

| 交易所     | Polygon支持 | 典型费用   |

## 获取MATIC用于gas费

### 为什么您需要MATIC

**区块链费用结构：**

- gas费必须以网络的原生货币支付
- Polygon的原生货币是MATIC
- 即使以USDC支付GPU租赁费用，交易本身也需要MATIC
- 没有MATIC = 无法发送交易

**所需金额很小：**

| 发送USDC     | 0.001-0.01 MATIC | $0.001-0.01 |

| 复杂交易     | 0.02-0.10 MATIC  | $0.02-0.10  |

**$1-2的MATIC可以覆盖数十笔交易。**您不需要太多，但需要一些。

## 将钱包连接到GPU租赁平台

您的钱包已充值。现在将其连接到GPU租赁平台。本节解释连接期间发生的情况以及您授予的权限。

### 理解钱包连接

当网站请求钱包连接时，您正在授予有限访问权限：

**连接允许：**

- 网站可以看到您的钱包地址
- 网站可以看到您的代币余额
- 网站可以请求交易（您必须单独批准）

**连接不允许：**

- 网站不能在没有您批准的情况下转移资金
- 网站不能自动签署交易
- 网站不能访问您的私钥或助记词

**钱包连接就像出示您的身份证**，而不是交出您的钥匙。网站了解您是谁（您的地址），但不能代表您行事。

### 连接到GPUFlow

**分步指南：**

1. 访问gpuflow.app
2. 确保MetaMask设置为Polygon网络
3. 点击"Connect Wallet"（通常在右上角）
4. MetaMask弹窗显示连接请求
5. 验证网站URL是否正确（防钓鱼）
6. 点击"Connect"
7. 您的钱包地址现在显示在网站上

**发生了什么：**

- GPUFlow现在可以显示您的USDC余额
- GPUFlow可以准备交易供您签署
- GPUFlow没有明确批准无法访问您的资金

### 理解交易批准

连接后，您将遇到两种类型的交易：

**类型1：代币批准**

在平台使用您的USDC之前，您必须为其智能合约"批准"代币：

```
MetaMask弹窗显示：
"授予访问您的USDC的权限？"
请求者：gpuflow.app
合约：0x7a3B...
```

**这意味着什么：**

- 您允许指定的合约从您的钱包转移USDC
- 只有那个特定的合约才能使用此权限
- 您可以稍后撤销此权限
- 通常批准"无限"金额以方便（避免重复批准）

**类型2：交易确认**

当您实际存款、租赁或提款时：

```
MetaMask弹窗显示：
"确认交易"
操作：向托管存入50 USDC
gas费：0.02 MATIC（~$0.02）
```

**这意味着什么：**

- 您正在授权特定金额的特定操作
- 显示的gas费将从您的MATIC余额中扣除
- 点击"Confirm"在区块链上执行交易
- 一旦确认，交易不可逆转

### 安全最佳实践

**在连接到任何网站之前：**

1. **验证URL：**仔细检查拼写（gpuflow.app对比gpufl0w.app）
2. **检查证书：**查找https://和有效证书
3. **为合法网站添加书签：**避免搜索和点击广告

**批准交易时：**

1. **阅读您正在批准的内容：**代币金额、接收合约
2. **验证gas费是否合理：**Polygon交易应低于$0.10
3. **检查网络：**确认您在预期的网络上（Polygon）
4. **不要着急：**花时间验证详细信息

**定期维护：**

1. **审查批准：**访问revoke.cash查看所有代币批准
2. **撤销未使用的批准：**移除不再使用的平台的权限
3. **使用单独的钱包：**考虑使用有限资金的专用钱包进行GPU租赁

### 连接到其他平台

**使用加密货币的Vast.ai：**

1. 创建Vast.ai账户（需要电子邮件）
2. 导航到Billing
3. 点击"Add Credit" → "Cryptocurrency"
4. 选择加密货币类型
5. 通过CoinPayments网关完成支付
6. 不需要直接钱包连接（通过网关支付）

**使用加密货币的RunPod：**

1. 创建RunPod账户
2. 导航到Billing
3. 点击"Add Funds" → "Crypto"
4. 选择加密货币类型
5. 通过Coinbase Commerce完成支付
6. 不需要直接钱包连接

**注意：**Vast.ai和RunPod使用支付网关而不是直接钱包连接。您向提供的地址发送加密货币，而不是将钱包连接到他们的网站。这与GPUFlow的直接钱包集成模式不同。

有关不同平台如何处理加密货币支付的全面比较，请参阅我们的[使用加密货币租赁GPU完整指南](/zh_cn/rent-gpu-with-crypto/)。

### 每个权限的含义

| 权限请求             | 允许的内容                 | 风险级别   |
| -------------------- | -------------------------- | ---------- |
| "Connect wallet"     | 查看您的地址和余额         | 非常低     |

| "Unlimited approval" | 合约可以转移任意数量的代币 | 中         |

**应拒绝的危险信号：**

- 对您不打算使用的代币的批准请求
- 与您预期不同的交易金额
- gas费远高于正常水平（错误的网络？）
- 请求助记词（绝对不合法）
- 为不熟悉的目的签署消息

## 常见问题故障排除

即使是简单的设置也会遇到问题。本节介绍新用户面临的最常见问题。

### "Transaction Failed"错误

**症状：**尝试确认后，MetaMask中的交易显示失败状态。

**常见原因和解决方案：**

**gas的MATIC不足：**

- 错误消息通常提到"insufficient funds for gas"
- 您有USDC但没有MATIC
- 解决方案：使用水龙头或交易所提款获取MATIC

**代币余额不足：**

- 尝试发送的USDC超过您拥有的数量
- 记住：gas费减少的是可用的MATIC，而不是USDC
- 解决方案：减少交易金额或添加更多代币

**滑点太低（用于兑换）：**

- 价格在报价和执行之间发生变化
- 解决方案：在兑换设置中增加滑点容差（通常1-3%就足够了）

**Nonce问题：**

- 之前的交易卡住，阻止后续交易
- 解决方案：在MetaMask设置中，Advanced → Reset Account（不会丢失资金）

### 资金没有出现

**症状：**您发送了资金，但钱包显示零余额。

**按以下顺序检查：**

**1. 选择了正确的网络吗？**

- 最常见的问题：当资金在Polygon上时查看Ethereum（反之亦然）
- 点击网络下拉列表，切换到正确的网络
- 相同的地址，不同的网络 = 不同的余额

**2. 交易真的完成了吗？**

- 检查发送方（交易所、其他钱包）
- 找到交易哈希
- 在适当的区块浏览器上搜索：
  - Polygon：polygonscan.com
  - Ethereum：etherscan.io
- 确认交易显示"Success"状态

**3. 需要导入代币吗？**

- 有些代币不会自动出现
- 在MetaMask中点击"Import tokens"
- 粘贴代币合约地址：
  - Polygon上的USDC：`0x3c499c542cEF5E3811e1192ce70d8cC03d5c3359`
- 导入后代币应该出现

**4. 发送到错误的网络了吗？**

- 发送到Ethereum但期望在Polygon上？
- 资金没有丢失——它们在不同的网络上
- 将MetaMask切换到您实际发送到的网络
- 如果需要，将资金跨链到预期的网络

### MetaMask连接问题

**症状：**网站无法连接到MetaMask或连接断开。

**解决方案：**

**刷新并重试：**

- 刷新网页
- 再次点击"Connect Wallet"
- 在MetaMask弹窗中批准

**检查MetaMask是否已解锁：**

- 点击MetaMask图标
- 如果已锁定则输入密码
- 重试连接

**禁用冲突的扩展：**

- 其他钱包扩展可能会干扰
- 临时禁用Phantom、Coinbase Wallet等
- 仅在MetaMask活动的情况下重试连接

**尝试不同的浏览器：**

- 存在特定于浏览器的问题
- 尝试Chrome、Firefox或Brave
- 如果问题持续存在，重新安装MetaMask扩展

**清除MetaMask连接：**

- 在MetaMask中：Settings → Connections
- 找到有问题的网站
- 点击"Disconnect"
- 重新连接

### 错误网络错误

**症状：**网站显示"Wrong network"或"Please switch to Polygon"。

**解决方案：**

1. 在MetaMask中点击网络下拉列表
2. 选择"Polygon Mainnet"
3. 如果Polygon未列出，添加它（参见之前的部分）
4. 刷新网站页面
5. 如有需要重新连接钱包

**大多数GPU平台会显示提示**以自动切换网络。出现提示时，在MetaMask弹窗中点击"Switch Network"。

### Gas估算失败

**症状：**尝试交易时MetaMask显示"Gas estimation failed"。

**常见原因：**

**合约拒绝交易：**

- 操作的代币余额不足
- 尝试与不兼容的合约交互
- 平台维护或问题

**网络拥堵：**

- 在Polygon上很少见但可能发生
- 等待几分钟后重试

**解决方法：**

1. 验证您有足够的余额进行预期操作
2. 验证您在正确的网络上
3. 尝试较小的交易金额
4. 检查平台状态（Discord、Twitter）是否有已知问题
5. 如果怀疑网络拥堵，稍后再试

### 待处理交易卡住

**症状：**交易长时间显示"Pending"（在Polygon上超过5分钟）。

**正常的待处理时间：**

- Polygon：5-30秒
- Ethereum：30秒 - 5分钟
- 更长时间表明有问题

**解决方案：**

**加速交易：**

1. 在MetaMask中点击待处理的交易
2. 点击"Speed Up"
3. 增加gas价格
4. 确认新交易
5. 更高的gas替换卡住的交易

**取消交易：**

1. 点击待处理的交易
2. 点击"Cancel"
3. 确认取消（需要gas费用）
4. 原始交易被放弃

**最后手段 - 重置账户：**

1. MetaMask Settings → Advanced
2. 点击"Reset Account"
3. 清除交易历史和nonce
4. 不影响余额
5. 重试原始交易

### 恢复场景

**丢失密码（但有助记词）：**

1. 卸载MetaMask
2. 重新安装MetaMask
3. 选择"Import using Secret Recovery Phrase"
4. 输入您的12个单词
5. 创建新密码
6. 钱包连同所有资金一起恢复

**丢失助记词（但可以访问MetaMask）：**

1. 在您仍有访问权限时打开MetaMask
2. Settings → Security & Privacy
3. 点击"Reveal Secret Recovery Phrase"
4. 输入密码
5. 这次正确记下助记词
6. 安全存储

**密码和助记词都丢失：**

- 资金永久丢失
- 无法恢复
- 任何支持团队都无法提供帮助
- 这就是为什么助记词备份至关重要

## 常见问题

### 为什么GPU租赁需要MetaMask？

MetaMask是一个加密货币钱包，可以存储您的资金并允许您与基于区块链的平台互动。使用加密货币支付的GPU租赁平台需要钱包来持有您的资金并批准交易。当您在GPUFlow等平台上租赁GPU时，您的付款进入智能合约托管——MetaMask是您授权该存款并稍后收回未使用资金的方式。推荐使用MetaMask是因为它是兼容性最广的钱包，几乎适用于所有接受加密货币的平台。它作为浏览器扩展安装，大约需要5分钟设置，并且免费使用。

### 为什么GPU租赁使用Polygon而不是Ethereum？

Polygon提供显著降低的交易费用——每笔交易$0.001-0.05，而Ethereum为$1-10或更多。对于GPU租赁，您可能需要存入资金、开始租赁、停止租赁和提取剩余余额（最少四笔交易），Ethereum的gas费很容易超过短期GPU租赁的成本。在RTX 4090上进行2小时租赁花费$1.20，需要$10-20的Ethereum gas费，这在经济上是不合理的。在Polygon上相同的工作流程总费用不到$0.10。像GPUFlow这样的GPU平台使用Polygon，正是因为低费用使小额和中等交易变得实用。

### 什么是助记词，为什么它如此重要？

助记词（也称为恢复短语或秘密恢复短语）是一个由12或24个单词组成的序列，可以完全访问您的加密货币钱包。这些单词按其确切顺序数学推导出您的私钥。任何拥有这些单词的人都可以从世界任何地方的任何设备访问钱包中的所有资金。没有密码重置、身份验证或支持团队可以在您丢失助记词时提供帮助——如果设备发生故障，这是恢复钱包的唯一方法。您必须将其写在纸上，绝不要以数字方式存储，并将其保存在安全的物理位置。设备故障时丢失助记词意味着所有资金的永久、不可恢复的损失。

### gas费需要多少MATIC？

很少。Polygon交易费用约为每笔交易$0.001-0.05，因此$0.50-2.00的MATIC（按当前价格约为0.5-2 MATIC）为50-200+笔交易提供足够的gas。这涵盖了包括存款、租赁和提款在内的几个月的典型GPU租赁使用。即使您用USDC支付GPU租赁费用，您也特别需要MATIC用于gas，因为区块链交易费用必须以网络的原生货币支付。如果您刚开始，从水龙头获得0.5 MATIC（免费）足以进行初始交易，如果需要，您可以稍后获得更多。

### 我可以使用MetaMask以外的钱包吗？

可以。替代方案包括Rainbow（出色的移动体验）、Coinbase Wallet（方便Coinbase用户）、Rabby（开发者友好功能）、Trust Wallet（专注于移动端）以及具有浏览器扩展集成的硬件钱包（如Ledger）。GPU租赁平台通常支持任何与WalletConnect协议兼容或直接Ethereum兼容浏览器扩展连接的钱包。推荐初学者使用MetaMask，因为它有广泛的文档、跨平台最广泛的兼容性、最大的用户社区以提供故障排除帮助，以及最长的记录。一旦对加密货币感到舒适，您可以探索更符合您偏好的替代方案。

### 如果我将资金发送到错误的网络会怎样？

发送到错误网络的资金不会丢失——它们只是在与预期不同的网络上。您的钱包地址在Ethereum和Polygon上是相同的，因此发送到任一网络上您地址的资金都属于您。如果您在Ethereum上发送了USDC但期望在Polygon上：将MetaMask切换到Ethereum Mainnet，您将在那里看到资金。要将它们移至Polygon，使用跨链桥（portal.polygon.technology），这需要Ethereum gas费（$5-15）。错误会花费额外的费用和时间，但不会导致资金损失。在确认任何发送交易之前，始终验证选定的网络。

### 我如何知道连接钱包到网站是否安全？

通过多项检查验证安全性。首先，确认URL拼写正确——钓鱼网站使用看起来相似的域名（gpufl0w.app而不是gpuflow.app）。其次，检查HTTPS和有效的安全证书。第三，为合法网站添加书签并使用书签而不是每次搜索。第四，在连接之前通过社区渠道（Discord、Reddit、Twitter）研究平台。当连接弹窗出现时，MetaMask显示请求网站——验证这是否符合您的预期。请记住，钱包连接只会透露您的地址和余额；实际的资金转移需要单独批准。在新平台上从小额开始，直到您通过成功的交易验证了合法性。

### "批准"代币是什么意思？

代币批准授予特定智能合约从您的钱包转移特定代币的权限。这与实际转移资金是分开的。当您向GPU租赁平台存入USDC时，会发生两笔交易：首先，您批准平台的托管合约访问您的USDC；其次，您确认实际存款。由于区块链代币的工作方式，需要批准——合约不能在没有明确许可的情况下从您的钱包中提取代币。您可以批准有限金额（耗尽时必须重新批准）或无限金额（方便但需要信任合约）。可以随时在revoke.cash等网站上审查和撤销批准。

### 为什么我的交易失败了？

交易失败有几个常见原因。gas的MATIC不足是最常见的——即使在USDC中交易，您也需要MATIC来支付费用。代币余额不足（尝试发送超过您拥有的数量）也会导致失败。与合约交互不正确（错误的网络、不兼容的参数）会触发失败。网络拥堵偶尔会导致问题，尽管这在Polygon上很少见。MetaMask中的错误消息通常指示原因。首先检查MATIC余额（新用户最常见的问题），然后验证您在正确的网络上，然后尝试减少交易金额。

### 如果我丢失密码，能恢复钱包吗？

可以，如果您有助记词。丢失MetaMask密码意味着您无法在该特定设备上访问钱包，但助记词提供完全恢复。卸载MetaMask，重新安装，选择"Import using Secret Recovery Phrase"，输入您的12个单词，然后创建新密码。您的钱包——包括所有余额——完全恢复。但是，如果您同时丢失密码和助记词，则无法恢复。没有公司、支持团队或技术流程可以在没有助记词的情况下恢复钱包。这就是为什么在向钱包添加任何资金之前，安全的助记词备份绝对至关重要。

### Polygon上的交易需要多长时间？

在正常条件下，Polygon交易通常在2-5秒内确认。您会看到交易在MetaMask中短暂显示为"pending"，然后变为"confirmed"。偶尔在高网络使用期间，确认可能需要15-30秒。如果交易保持待处理状态超过5分钟，则可能出现了问题——gas价格不足、网络问题或nonce问题。您可以通过点击待处理交易并选择"Speed Up"来"加速"MetaMask中卡住的交易，这会以更高的gas重新提交。与Ethereum主网（30秒到几分钟）和比特币（10-60分钟）相比，Polygon的速度在实际用途上基本上是即时的。
